# TrustGuard Project Notes

## Project Title

TrustGuard: A Trust-Aware Multi-Agent Cyber Defense Architecture for Secure Agentic AI Systems

## Team Members

* Syed Farhaan
* Likhitha

## Project Status

Research Planning Phase

## Objective

To design and develop a trust-aware multi-agent framework capable of defending agentic AI systems against prompt injection, memory poisoning, and adversarial manipulation attacks.

## Planned Features

* Dynamic Trust Propagation Mechanism (DTPM)
* Memory Provenance Graph
* Consensus-Based Decision Verification
* Adaptive Trust Decay Strategy
* Multi-Agent Threat Detection
